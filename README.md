This project is a todo list maker that keeps the data in the localhost.

# Features
- Responsive layout;
- Simple UI;
- Active & completed tasks archive view;

# Technologies
- HTML5
- CSS3
- javascript
- ES6
- Font Awesome
- ReactJS

# Status
- done

# Running it locally 

### `yarn install`

### `yarn start`
